
# Himanshu Kumar — Cybersecurity Portfolio

## How to Host
### ✅ GitHub Pages
1. Create a repository named: `himanshu-portfolio` or anything you like
2. Upload all files from this zip
3. Go to **Settings → Pages → Source: main branch**
4. Save — your site will be live at:
   `https://devil136-star.github.io/Himanshu-s-Portfolio/`

### ✅ Netlify Deployment
1. Go to https://app.netlify.com
2. Click **New site → Deploy manually**
3. Upload this folder
4. Site goes live instantly

### ✅ Vercel Deployment
1. `npm i -g vercel`
2. `vercel deploy` inside this folder
